# 深拷贝

## 项目说明

实现一个深拷贝函数，支持普通对象、数组、Map、Set，并通过 WeakMap 处理循环引用。

## 涉及知识点

- 深拷贝与浅拷贝
- 递归
- `WeakMap`
- 循环引用处理
- `Map` / `Set` 拷贝
- Jest 单元测试

## 文件说明

| 文件 | 说明 |
|---|---|
| `deepclone.ts` | 深拷贝实现 |
| `_deepclone.ts` | 另一版实现记录 |
| `deepclone.test.ts` | 单元测试 |

## 快速启动

```bash
npm run demo -- deep-clone
```

也可以使用编号启动：

```bash
npm run demo -- 27
```

## 核心思路

对于基本类型和 `null`，直接返回原值。对于引用类型，递归拷贝内部属性。

为了处理循环引用，使用 `WeakMap` 记录已经拷贝过的对象。如果再次遇到同一个对象，直接返回缓存中的拷贝结果。

## 总结

深拷贝是理解引用类型、递归遍历和循环引用处理的重要实验。
