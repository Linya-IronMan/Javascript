# 数组扁平化

## 项目说明

实现数组扁平化函数，包含浅层扁平化和递归深层扁平化。

## 涉及知识点

- 数组遍历
- `Array.isArray`
- 递归
- `concat` 与展开运算符
- Jest 单元测试

## 文件说明

| 文件 | 说明 |
|---|---|
| `index.ts` | 数组扁平化实现 |
| `index.test.ts` | 单元测试 |

## 快速启动

```bash
npm run demo -- flatten
```

也可以使用编号启动：

```bash
npm run demo -- 20
```

## 核心思路

浅层扁平化只处理一层嵌套数组，深层扁平化需要在遇到数组元素时递归处理，直到所有元素都变成非数组值。

## 测试说明

测试覆盖：

- 空数组
- 非嵌套数组
- 一级嵌套数组
- 多级嵌套数组

## 总结

数组扁平化是理解递归、数组结构处理和手写工具函数的基础练习。
